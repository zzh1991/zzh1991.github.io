windows 下解压 Linux、macOS 压缩格式

  • 在线的转换格式网站
  • 7zip manager 工具选项里勾选 tgz 等格式
  • 减小切换系统，方便直接转换

各编程语言的开发文档及 API

  • dash 整合的文档下载地址

  • Zeal (可以整合 IDE 一起使用，其自带一些文档)
  • DevDocs: 在线 API 网站，唯一不好的就是没有 JAVA 的文档

PDF 与 MS Office 以及图片格式的互相转换

  • SmallPDF
  • IlovePDF
