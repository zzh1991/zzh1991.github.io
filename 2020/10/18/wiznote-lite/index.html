为知笔记 Lite 使用指南

发表于2020-10-18|更新于2021-01-09|note
|阅读量:

这篇文章主要介绍一下最新的为知笔记 Lite 编辑器的使用指南。由于官方并没有详细的使用说明，本人在尝试后总结了一些使用经验。

标题

目前一共有 5 级标题，点击相应的标题可以进行标题级别的更改，同时也可以转化为普通段落或者引用。

使用 + 或者 / 快速插入

输入 + 或者 / 可以快速插入相应的板块，包括链接，代办事项，表格，图片已经代码块。

代码块

代码的使用方式跟 Markdown 的语法是一模一样的，只需要选择合适的语言进行展示即可。

hello world

public class HelloWorld {

}

mermaid

graph TD;
A-->B;
A-->C;
B-->D;
C-->D;

公式

使用公式以 $$ 开始和结尾，中间就是常见的 Latex 的公式语法了。

其中单 $ 是行间公式，而双 $$ 为公式块表示。

x=y{x=y}x=y

x2=y∗z{x^2=y * z} x2=y∗z

思维导图

其实思维导图，其实就是为知笔记里面的大纲模式。

- spring
    - spring ioc
        - bean factory
        - application context
            - file system xml
            - class path xml
            - web xml
    - bean life cycle
        - initialize
            - initializing bean
        - destroy
            - disposal bean

快捷键

标题

command + option + [12345]可以控制标题级别

加粗

command + B 可以用来加粗

斜体

command + I 可以倾斜字体

有序列表

command + O 可以编写有序的列表

  1. 第一个
  2. 第二个

无序列表

command + L可以编写无序的列表

  • 一个
  • 两个

表格

command + M 可以用来生成表格

col1 col2 col3

代码块

command + U 可以用来生成代码块

echo "hello world"

行间代码块

command + G 可以生成行间代码 hello world

代办事务

command + J 可以生成代办事项

  • [X] 探索为知笔记 Lite 的快捷键
  • [ ] 使用说明

最新快捷键

引用

可以使用 option + command + Q 来生成引用模块

不变的快捷键

标题、加粗、斜体的快捷键都没有改变

突然失效的快捷键

  • 无序列表

有序列表

  1. 使用 option + command + O 生成有序列表

表格

使用 shift + command + T

c1 c2 c3

行间代码块

使用 command + ~ 可以生成行间代码

hello world

代码块

使用 option + command + C 可以生成代码块

public static void main(String[] args) {

}

代办事项

option + command + X 可以用来生成代办事项

  • [x] 完善最新的快捷键说明

链接

使用 command + L 可以生成链接

百度

图像

使用 shift + command + I 可以插入图片

公式

使用 option + command + M 生成公式

x2+y2=zx^2 + y^2 = z x2+y2=z
