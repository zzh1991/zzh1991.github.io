Intellij IDEA 完美配置，让你爱不释手

发表于2020-09-07|更新于2021-10-30|Java
|字数总计:662|阅读时长:2分钟|阅读量:

作为每天写代码的工具，配置一个称心如意的 Intellij IDEA 就显得尤为重要了。

外观篇

每天面对着自己的喜欢的风格编写代码是一件非常惬意的事情。

主题篇

个人比较喜欢暗黑的主题，对眼睛会比较友好。另外可以安装 Material Theme UI 主题插件

字体篇

这里使用的是 Color Scheme Font，它覆盖了默认的 Font 字体，字体采用的是 Fira Code。由于我喜欢较大的字号，我选择了 16 号，字号大小还是因人而异的。

括号篇

  • Rainbow Brackets：让你拥有五颜六色的括号（每一对括号的颜色都是不一样的）

效率篇

效率第一人 Lombok

记得开启注解处理哦，不然在代码里会出现很多错误。

Live Template

快速生成代码模版

Java 篇

字符处理篇

使用 String Manipulation 插件可以进行各种字符的变换。

该插件的功能非常丰富，值得学习使用。

重构篇

学习如何使用重构相关的快捷键，可以进行快速重构

代码提示篇

除了自带的自动补全之外，还可以使用以下的插件增强自动补全功能。

  • Codota

快捷键篇

  • Key Promoter X
  • AceJump：快捷键为 control + ;

Override

macOS 的快捷键为 Control + O

Generate

macOS 的快捷键为 Command + N

翻译篇

在取名和阅读源码的过程中，不可避免会碰到不认识的单词，那这个时候快速的翻译工具可以大大提高我们的编程效率。

代码规范篇

  • Alibaba Java Coding Guidelines
  • Git Commit Template

字节码篇

  • jclasslib Bytecode viewer

针对 class 文件，选择 View tab，可以看到查看字节码的选项

依赖管理篇

推荐使用 Maven Helper

生成序列化版本号

  • GenerateSerialVersionUID（可以在 Generate 中找到）

JSON to Object

  • GsonFormat

图像篇

类 UML 图

右键即可调出 Diagrams 功能

方法调用时序图

安装 Sequence Diagram 后，选择一个方法名，右键即可

思维导图

配置篇

自动导入

方法分割线

自动提示不区分大小写

演出模式

使用演出模式可以拥有极佳的阅读代码的体验。可以使用 command + e 快速切换文件列表，查找已经打开的文件；使用 shift + shift 快速查找文件。

文档篇

JavaDoc

技巧篇

行尾添加分号

使用 shift + command + enter 可以快速补全分号

彩蛋

最新的 IDEA 版本可以在这里看到哦。
